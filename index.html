Welcome to PaintOver, a colorful, musical game for all to enjoy!

The Game

The objective of the game is to flood the whole board into a single color
  1. The top-left square is "flooded"
  2. Select any square on the board, and all "flooded" squares will change to that color
  3. Any squares adjacent to a "flooded" square will become flooded if it matches the color you chose

The Music

The controls to the left of the board adds music to the game
  • Click the Music button to begin the sequencer
  • Press the toggle color buttons to "activate" the color
  • The Painter button will let you color any square with the selected color
  • The WackyPainter button will fill in a random color for every square you mouse over
